Upload a CSV file to validate transaction data.

📌 Supported File Format

This platform accepts only .csv files containing the following columns exactly as specified:

  • order_id
  • order_date
  • customer_name
  • phone_number
  • country
  • product_id
  • product_name
  • quantity
  • unit_price
  • payment_mode
  • payment_status
  • amount_paid
  • transaction_id

Please ensure that:

  • The uploaded file is in .csv format.
  • Column names match exactly, including underscores and spelling.
  • Missing, extra, or incorrectly named columns may cause the file to be rejected.
  • Files with a different schema are currently not supported.
  • The first upload after inactivity may take around 20–30 seconds as the backend service hosted on Render needs to wake up.
